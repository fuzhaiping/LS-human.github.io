Welcome to LS-human Pages.

LS-human dataset was collected using 5 RGBD cameras, capturing RGBD data of human scenes from multiple perspectives. We simulate a 3D live streaming scenario where the characters interact with the audience, incorporating vivid facial expressions and challenging hand gestures in the video. This dataset can address the current lack of RGBD human scene datasets. Additionally, LS-human can be widely used for training tasks such as rendering new views under sparse viewpoints.
